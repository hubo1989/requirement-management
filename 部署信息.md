## 部署步骤

1. 使用GitHub CLI创建仓库并推送代码
   ```bash
   gh repo create requirement-management --public --description "需求管理系统 - 包含银行数据看板PRD和商户维度分析功能" --source=. --remote=origin --push
   ```

2. 使用Vercel CLI部署项目
   ```bash
   vercel --prod
   ```

## 注意事项

- 由于文件路径包含中文字符，直接访问可能需要使用URL编码
- 建议通过导航页面访问各个功能模块
